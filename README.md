### 學習技術

##### 前端

- html、css、javascript
  - 基本標籤`<a href>`、`<input type>`、`<form>`
- jsp、jQuery、JSTL標籤庫、EL運算式
  - `<c:if test="">`、`${key1}`、綁定點擊事件
- Ajax
  - `$.getJSON()`輸出響應

##### 後端

> 不依賴框架完成http請求與響應的操作模型

- Servlet、Filter、Listener、Cookie、Session、JSON
- web容器的運作與部屬
- MVC與三層架構搭建javaweb專案
